# Prompt: 30/60/90 Plan

## Purpose
Build a comprehensive 30/60/90 day plan for any project, role transition, or initiative — combining horizon-based planning with Pareto prioritization and OKR-based goals.

## When to Use
- Starting a new project or role
- Quarterly planning sessions
- When priorities are unclear and a team needs alignment

## Variables

| Variable | Description | Example |
|---|---|---|
| `{{context}}` | The project, initiative, or role being planned | "Launching an AI workflow for customer support" |
| `{{vision}}` | What success looks like at 90 days | "AI resolves 60% of Tier-1 support tickets automatically" |
| `{{stakeholders}}` | Who is involved and their roles | "CTO (sponsor), support lead (owner), 3 support reps (users)" |
| `{{constraints}}` | Known limitations | "No new budget; must use existing tooling" |

## Prompt

```
You are a strategic planning facilitator applying the 30/60/90 planning framework.

Context: {{context}}
Vision: {{vision}}
Stakeholders: {{stakeholders}}
Constraints: {{constraints}}

Build a comprehensive 30/60/90 plan:

## 1. Foundation
- Refined vision statement (one precise sentence)
- 3–5 Objectives (measurable, time-bound)
- 2–3 Key Results per objective
- Stakeholder alignment map (who must be aligned, by when)

## 2. Planning Matrix

For each time horizon (30/60/90), identify key activities across these categories:

| Category | 30 Days (Immediate) | 60 Days (Soon) | 90 Days (Long Term) |
|---|---|---|---|
| Learn | | | |
| Analyze | | | |
| Implement | | | |
| Relationships | | | |

## 3. Pareto Filter
Within each time horizon, identify the top 20% of activities that will deliver 80% of the value. Mark these as **Priority** items.

List:
- 30-day Pareto priority (1–2 items)
- 60-day Pareto priority (1–2 items)
- 90-day Pareto priority (1–2 items)

## 4. Value Stream Tagging
For each priority item, tag with primary value stream:
- 💰 Revenue Generation
- 🛡️ Risk Mitigation
- 💸 Cost Savings

## 5. Success Metrics
For each horizon, define:
- What does "done" look like?
- Which metric confirms progress?
- Expected Rate of Improvement (trajectory, not just endpoint)

## 6. Dependency Map
List: which 30-day tasks must complete before 60-day work begins? Which 60-day before 90-day?
```

## Expected Output
- Completed planning matrix across all 4 categories × 3 horizons
- Pareto priorities for each horizon
- Success metrics with measurable targets
- Dependency map for sequential work

## Tips
- If the 30-day column is overwhelming, the scope needs trimming — apply Pareto immediately
- Each horizon should have a clear "graduation criteria" — what exactly must happen to move to the next horizon
